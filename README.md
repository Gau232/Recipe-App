# Recipe App

This recipe web app is created using React.

## Demo

https://recipe-app-gau232.vercel.app/

## Features
- Recipe list
- Recipe categories
- Recipe details
- Search recipes
- Offline support

## Tech Stack

React JS, HTML and CSS

## Libraries/Packages used
- react splide
- react router dom
